# aws-module-ec2
Terraform module for creating a basic AWS EC2 instance with essential configurations. Designed for simplicity and reusability

### Structure

```bash
aws-module-ec2/
├── main.tf
├── variables.tf
├── terraform.tfvars
└── module/
    └── ec2/
        ├── ec2.tf          
        └── variables.tf
```
